# Dsquare Med-assist Website Files

This directory contains HTML templates for the Dsquare Med-assist website to be deployed on Hostinger.

## Files Structure

- `index.html` - Homepage
- `medobsmind.html` - MedObsMind LLMM page  
- `ddma.html` - DDMA (Dsquare Digital Medical Assistant) page
- `styles.css` - Stylesheet
- See `../WEBSITE_PROMPTS.md` for complete content and deployment instructions

## Deployment to Hostinger

1. Upload all files to `/public_html/` directory via File Manager or FTP
2. Configure domain and SSL
3. Test all pages and links
4. Update contact form with actual email addresses

For detailed instructions, see WEBSITE_PROMPTS.md
